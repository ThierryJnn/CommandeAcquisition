void speed ( int  digit)

Contrôle la vitesse du moteur.

Valeurs retournées
None
Paramètres
intdigit
Note
La valeur de la vitesse à régler est entre 0 et 1024, 0 pour un rapport cyclique de 0%, 512 pour un rapport cyclique de 50% et 1024 pour un rapport cyclique de 100%. Le moteur est à l'arret pour un rapport cylique de 50%.
